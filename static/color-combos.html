Preview writes the combination to this browser's localStorage (key wefeasto_colormix) — the same place the site's toggle keeps its shuffle — then opens the site; every page renders it until the toggle is switched off. The toggle itself re-shuffles randomly each time it's turned on, so to get back to a chosen combination, re-apply it here. Original palette = toggle off.
